# CLAUDE.md — AppEngine (this repo)

This repository **is** App Engine — the **factory** in Lincoln's connected ecosystem
(United Under God / Life Produces Life). It builds the other tools; it is **not** the product,
the mission, or the hub.

## Read first — the canonical source of truth (separate `life-produces-life` repo)

1. `life-produces-life/_SOURCE_OF_TRUTH/00_LIFE_PRODUCES_LIFE__MASTER.md` — the mission
2. `life-produces-life/_SOURCE_OF_TRUTH/03_APP_ENGINE__SOURCE_OF_TRUTH.md` — App Engine's scope, agents, boundaries

See `AGENTS.md` in this repo for the technical build guide (commands, env vars, working style).

## Rules
- Consume the philosophy; never invent or change it — ask Lincoln.
- Work only from the board: claim the next `ready` item, one branch + one PR, don't edit outside your item.
- **Build connected, not siloed** — ecosystem expressions use the shared Supabase identity; each
  customer-built app gets its own isolated Neon (see `03_APP_ENGINE` §7–8). One person, many
  relationships, one ecosystem.
- Lincoln approves merges to `main`, deploys, database changes, and paid resources.
- App Engine optimizes for **finished** tools, not impressive prototypes.
